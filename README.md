## Getting Started

### Prerequisites
- Python 3.7+ installed  
- `unittest` (built into Python)

### Running Tests
You can run all tests at once with:
```bash
python -m unittest
```
Or, if available, run the `run_all_tests.py` script which runs all test files and displays a status summary.
